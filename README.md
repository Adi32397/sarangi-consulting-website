# Sarangi Consulting - Website Prototype

A premium, high-fidelity corporate frontend website prototype designed for Sarangi Consulting. Built without frameworks, using only pure HTML, CSS, and JavaScript to deliver an executive, fast, and modern web experience.

## 🚀 Features

- **Single-Page Application (SPA) Routing:** Seamlessly switches between sections without page reloads.
- **Premium Aesthetics:** Inspired by top-tier consulting firms (McKinsey, Bain, BCG) with a custom color palette (Dark Green, Executive Black, White).
- **Smooth Animations:** 
  - Dynamic scroll reveals.
  - Interactive "crazy but formal" 3D flip cards in the *Why Choose* section.
  - A continuous, futuristic laser-drop animation in the *Growth Framework* timeline.
- **Fully Responsive:** Perfectly scaled for Desktop, Tablet, and Mobile devices.
- **Developer Showcase:** Built-in developer profile card.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom variables, grid/flexbox layouts, 3D transforms, keyframe animations, and glassmorphism.
- **Vanilla JavaScript:** SPA routing, Intersection Observer for scroll animations, and counter animations.
- **FontAwesome:** For vector icons.
- **Google Fonts:** Montserrat (Headings) & Inter (Body).

## 📂 Project Structure

```text
├── index.html     # Main HTML file containing all sections
├── style.css      # Custom styling, animations, and responsive media queries
├── script.js      # JavaScript for routing, scroll reveals, and counters
├── founder.jpg    # Image asset for the Founder & CEO
├── developer.jpg  # Image asset for the Developer
└── README.md      # Project documentation
```

## 💻 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Adi32397/sarangi-consulting-website.git
   ```
2. Navigate into the directory:
   ```bash
   cd sarangi-consulting-website
   ```
3. Open `index.html` in your favorite web browser (or use Live Server in VS Code for a better development experience).
