# WORKFLOW PHILOSOPHY

Every user interaction should guide the visitor toward one of the following outcomes:

1. Book Discovery Call

2. Purchase Startup Advisory

3. Complete Business Assessment

4. Download Resource

5. Contact Sarangi

6. Subscribe To Newsletter

7. Engage With AI Advisor

The website should not behave as a brochure.

It should function as a lead-generation and consulting conversion platform.
